# 機器學習分類

機器學習可以分為兩類
- Classification 分類
- Regression 回歸

回歸與分類都是屬於「監督式學習」

| 名詞       | 說明               |
| -------- | ---------------- |
| Dataset  | 資料集，模型學習用的資料     |
| Feature  | 特徵，拿來判斷的條件       |
| Label    | 標籤，也就是正確答案       |
| Training | 訓練，用資料讓模型學習      |
| Testing  | 測試，用沒看過的資料檢查模型表現 |
| Predict  | 預測，讓模型回答新資料的答案   |

## 回歸

預測連續數值，

| 問題           | 預測結果          |
| ------------ | ------------- |
| 散步多久會消耗多少熱量？ | 120 大卡、180 大卡 |
| 房子大小會影響房價多少？ | 800 萬、1200 萬  |
| 溫度和銷售量的關係？   | 賣出 50 杯、80 杯  |
| 停車多久要付多少錢？   | 30 元、60 元     |

### 線性回歸

線性迴歸是一種統計技術，用於找出變數之間的關係。在機器學習環境中，線性迴歸會找出特徵與標籤之間的關係。

舉例來說，假設我們想根據散步時間預測消耗多少熱量，並有以下資料集：

| 散步時間 （特徵） | 消耗熱量 （標籤） |
|-----------|-----------|
| 10        | 42        |
| 20        | 82        |
| 30        | 121       |
| 40        | 160       |
| 50        | 198       |
| 60        | 240       |

### 線性回歸方程式

以代數來說，模型會定義為 $y = ax + b$，其中

- $y$ 是消耗的熱量，也就是我們想預測的值。
- $a$ 是直線的斜率。
- $x$ 是散步時間，也就是輸入值。
- $b$ 是 y 截距。

*消耗的熱量 = a * 散步時間 + 截距*

在機器學習中，線性迴歸模型的方程式如下：

$y' = b + w_1x_1$

 - $y'$是預測標籤，也就是輸出內容。
 - $b$是模型的偏差。偏差與直線代數方程式中的 y 截距概念相同。在機器學習中，偏誤有時稱為$w_0$。偏誤是模型的參數，會在訓練期間計算。
 - $w_1$是特徵的權重。權重與代數方程式中直線的斜率概念相同。權重是模型的參數，會在訓練期間計算。
 - $x_1$是特徵，也就是輸入內容。

### 誤差與損失

| 名稱        | 白話意思              | 例子                 |
|-----------|-------------------|--------------------|
| 誤差 error  | 真實值和預測值差多少        | `真實值 - 預測值`        |
| 損失 loss   | 把誤差轉成一個可以讓模型學習的分數 | `誤差平方`、`誤差絕對值`     |
| 指標 metric | 用來評估模型表現的數字       | `MAE`、`MSE`、`RMSE` |

#### 誤差error

```預測值與真實值之間的差距```

#### 損失loss

```模型訓練時要最小化的目標```

#### 回歸模型的衡量標準

- MSE：平方絕對誤差
- MAE：平均平方誤差
- RMSE：均方根誤差

**MSE**：Mean Squared Error
$
\mathrm{MSE} = \frac{1}{n}\sum_{i=1}^{n}(y_i-\hat{y}_i)^2
$

**MAE**：Mean Squared Error
$\mathrm{MAE} = \frac{1}{n}\sum_{i=1}^{n}\left|y_i-\hat{y}_i\right|$

**RMSE**：Root Mean Squared Error

$
\mathrm{RMSE}
=
\sqrt{
\frac{1}{n}
\sum_{i=1}^{n}
(y_i-\hat{y}_i)^2
}
$

如何選擇？


| 情況             | 建議看        |
|----------------|------------|
| 想簡單知道平均差多少     | MAE        |
| 很在意大誤差         | MSE 或 RMSE |
| 想要跟原本單位一樣、方便解釋 | RMSE       |


### 練習

線性回歸

- 先算出xy平均
- 算斜率
$$
b_1=
\frac{\sum_{i=1}^{n}(x_i-\bar{x})(y_i-\bar{y})}
{\sum_{i=1}^{n}(x_i-\bar{x})^2}
$$

### 程式範例

```python
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_absolute_error, mean_squared_error
import numpy as np


# =========================
# 1. 準備資料
# =========================

# X 是特徵，也就是用來預測的資料
# 這裡的特徵是「散步時間」
#
# 注意：
# sklearn 的 X 必須是二維資料
# 所以不能寫成 [10, 20, 30, 40, 50, 60]
# 要寫成 [[10], [20], [30], [40], [50], [60]]
X = [
    [10],
    [20],
    [30],
    [40],
    [50],
    [60]
]

# y 是標籤，也就是真實答案
# 這裡的標籤是「消耗熱量」
y = [42, 82, 121, 160, 198, 240]


# =========================
# 2. 建立線性回歸模型
# =========================

# LinearRegression 是線性回歸模型
# 它會找出一條最適合資料趨勢的直線
model = LinearRegression()


# =========================
# 3. 訓練模型
# =========================

# fit() 代表讓模型學習資料
# 模型會根據 X 和 y 算出斜率與截距
model.fit(X, y)


# =========================
# 4. 取得斜率與截距
# =========================

# coef_ 是斜率 m
# intercept_ 是截距 b
m = model.coef_[0]
b = model.intercept_

print("線性回歸公式：")
print(f"y = {m:.3f}x + {b:.3f}")
print()


# =========================
# 5. 使用模型預測
# =========================

# predict() 會根據模型學到的線性公式產生預測值
y_pred = model.predict(X)


# =========================
# 6. 計算每一筆資料的誤差
# =========================

print("每一筆資料的預測結果")
print("-" * 80)
print("散步時間\t真實值\t預測值\t\t誤差\t\t絕對誤差\t平方誤差")
print("-" * 80)

for i in range(len(X)):
    walk_time = X[i][0]      # 散步時間
    true_value = y[i]        # 真實消耗熱量
    pred_value = y_pred[i]   # 模型預測消耗熱量

    # 誤差 = 真實值 - 預測值
    error = true_value - pred_value

    # 絕對誤差 = 誤差取絕對值
    absolute_error = abs(error)

    # 平方誤差 = 誤差平方
    squared_error = error ** 2

    print(
        f"{walk_time}\t\t"
        f"{true_value:.2f}\t"
        f"{pred_value:.2f}\t\t"
        f"{error:.2f}\t\t"
        f"{absolute_error:.2f}\t\t"
        f"{squared_error:.2f}"
    )


# =========================
# 7. 計算 MAE / MSE / RMSE
# =========================

# MAE：Mean Absolute Error
# 平均絕對誤差
# 代表模型平均差多少
mae = mean_absolute_error(y, y_pred)

# MSE：Mean Squared Error
# 平均平方誤差
# 會把大誤差放大懲罰
mse = mean_squared_error(y, y_pred)

# RMSE：Root Mean Squared Error
# MSE 開根號
# 單位會回到原本的單位，比 MSE 好解釋
rmse = np.sqrt(mse)


# =========================
# 8. 顯示整體誤差指標
# =========================

print()
print("整體誤差指標")
print("-" * 30)
print(f"MAE  平均絕對誤差：{mae:.3f}")
print(f"MSE  平均平方誤差：{mse:.3f}")
print(f"RMSE 均方根誤差：{rmse:.3f}")
```
