# 課程筆記與實作基地

歡迎來到這個課程儲存庫。

這裡會放上課用到的筆記、範例程式、操作流程與一些老師碎碎念。
如果你上課時聽懂了，很好，這裡可以幫你複習。
如果你上課時恍神了，也沒關係，這裡就是讓你回來補血用的。

## 這裡會放什麼？

這個儲存庫會依照課程進度慢慢更新，內容可能包含：

* 上課筆記
* 範例程式
* 實作流程
* 作業與練習
* 常見錯誤
* 老師覺得你可能會踩到的坑
* 一些「為什麼它又壞了」的解法

## 這份資料適合誰？

主要是給正在上課的同學使用。
當然，如果你只是路過，想看看這堂課到底在玩什麼，也歡迎。

適合以下幾種狀況：

* 上課想跟著做
* 課後想複習
* 程式跑不動想找範例
* 作業做到懷疑人生
* 忘記老師剛剛講什麼
* 想重新整理自己的學習進度

## 建議怎麼使用？

請不要只複製貼上，雖然我知道你很想。

比較推薦的方式是：

1. 先看這個主題要做什麼
2. 跟著筆記操作一次
3. 把範例程式跑起來
4. 嘗試改一點東西
5. 看它爆炸
6. 回來找錯誤原因
7. 修好它
8. 恭喜，你真的學到了

## 學習小提醒

寫程式或學工具的時候，遇到錯誤是正常的。
沒有錯誤才比較可怕，因為你可能根本還沒開始。

所以看到紅字不要慌，先深呼吸，然後檢查：

* 檔案路徑是不是錯了
* 套件有沒有安裝
* 變數名稱有沒有打錯
* 縮排是不是怪怪的
* 是不是少打一個括號
* 是不是其實你剛剛沒有存檔

大部分的錯誤，通常都不是電腦討厭你。
只是它很誠實。

## 更新方式

這個儲存庫會依照上課進度更新。
有時候會新增範例，有時候會修正內容，有時候會補上課堂中突然想到的東西。

所以如果你發現內容跟之前不太一樣，不用緊張，代表它有在成長。

## 最後

希望這份資料可以幫助你在課堂上跟得上，課後也能自己練習。
學習不是一次就全部懂，而是一次比一次更不怕錯。

這份資料會持續更新，也可能會有不小心寫錯、漏寫，或是範例程式沒有照顧到所有情況的地方。
如果你發現任何錯誤，或是有更好的建議，歡迎提出來指正。

看到錯誤不用客氣，歡迎救救老師，也救救其他同學。

慢慢來，會進步的。
