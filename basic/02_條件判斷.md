# 條件判斷

## if/else/elif

```python
score = 85

if score >= 90:
    print("等級：A")
elif score >= 80:
    # 只要符合條件，就會執行這個區塊，並跳過後面的 elif 與 else
    print("等級：B")  
elif score >= 70:
    print("等級：C")
else:
    print("等級：F")
```

> 注意順序，如果用大於，要由大比到小；反之用小於，就由小比到大

## match

```python
day = 6

match day:
    case 0:
        print('星期日')
    case 1:
        print('星期一')
    case 2:
        print('星期二')
    case 3:
        print('星期三')
    case 4:
        print('星期四')
    case 5:
        print('星期五')
    case 6:
        print('星期六')
    case _:
        print('錯誤')

x ='123a'
print(x.isdigit() is False)
```
