# List 串列

## 串列基礎

### 建立串列

```python
lt1 = ['apple','banana','mango']
print(lt1)
# ['apple','banana','mango']
```

### 索引

串列跟文字可以使用索引取值，第一筆資料索引為`0`

```python
print(lt1[0])
print(lt1[1])
# apple
# banana
```

### 負數索引

負數索引從最後一筆資料開始算，最後一筆資料索引為`-1`

```python
print(lt1[-1])
```

### 建立空串列

```python
lt1 = list()
lt2 = []
```

### 使用迴圈顯示串列內容

```python
for x in lt1:
    print(x)
```

若同時需要串列內容以及索引值，可以使用`enumerate()`

```python
for i,x in enumerate(lt1):
    print(i,x)

# (索引,內容)
```

## 串列方法

### 新增

- `append()`
- `insert()`
- `extend()`

```python
# append() 新增項目
fruits.append('orange')

# insert() 在指定位置新增項目
fruits.insert(0,'orange')

# extend() 新增可迭代物件
fruits.extend(['orange', 'kiwi','orange'])
```

### 刪除

- `remove()`
- `pop()`
- `del`
- `clear()`

```python
# remove() 移除項目
fruits.remove('orange')

# pop() 移除最後一個或指定項目
fruits.pop(0)

# del 透過索引移除項目
del fruits[-1]

# clear() 清空串列
fruits.clear()
```

### 修改

```python
# 將索引2的資料修改成芒果
fruits[2] = '芒果'
```

### 查詢與計算

- `index()`
- `count()`
- `len()`
- `in`

```python
# index()
# 判斷資料位置，如果存在回傳索引；如果不存在則會報錯
q = fruits.index('melon')

# count()
# 計算資料數量，回傳資料個數，不存在回傳0
q2 = fruits.count('melon')
print(q2)

# len()
# 計算串列長度
print(len(fruits))

# in
# 判斷資料是否存在，回傳布林值
print('orange' in fruits)
print('melon' in fruits)
```

### 排序

- `sort()`
- `reverse()`
- `sorted()`
- `reversed()`

#### `sort()`

```python
# 遞增排序
fruits.sort()

# 遞減排序
fruits.sort(reverse=True)
```

在預設的狀態下，如果同時有大寫跟小寫的資料，會先排序大寫再排序小寫，如果要避免該狀況，可以使用`key=str.lower`參數排除大小寫

```python
# 全部使用小寫作排序
fruits.sort(key=str.lower)
```

#### `reverse()` 反轉

```python
fruits.reverse()
```

`sort()`跟`reverse()`都會改變原始串列<br>如果不想改變可以使用`sorted()`跟`reversed()`

```python
fruits_sorted = sorted(fruits, key=str.lower)
print(fruits_sorted)

fruits_reversed = reversed(fruits)
for item in fruits_reversed:
    print(item)
```
