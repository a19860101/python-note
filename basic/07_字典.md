# Python Dictionary（字典）基礎與常用方法

## 什麼是 Dictionary？

Dictionary（簡稱 `dict`）是一種使用 **Key（鍵）** 與 **Value（值）** 配對儲存資料的資料型態。

例如：

| Key（鍵） | Value（值） |
|-----------|-------------|
| name | 小明 |
| age | 18 |
| height | 175 |

```python
student = {
    "name": "小明",
    "age": 18,
    "height": 175
}
```

---

# 建立 Dictionary

## 方法一：使用 `{}`（最常用）

```python
student = {
    "name": "小明",
    "age": 18,
    "score": 95
}

print(student)
```

輸出

```python
{'name': '小明', 'age': 18, 'score': 95}
```

---

## 方法二：使用 `dict()`

```python
student = dict()

student["name"] = "小明"
student["age"] = 18

print(student)
```

---

# 取得資料

透過 Key 取得 Value。

```python
student = {
    "name": "小明",
    "age": 18
}

print(student["name"])
```

輸出

```text
小明
```

---

# 修改資料

```python
student["age"] = 20

print(student)
```

---

# 新增資料

若 Key 不存在，會直接新增。

```python
student["score"] = 100

print(student)
```

---

# 刪除資料

## 使用 `del`

```python
del student["age"]
```

---

## 使用 `pop()`

刪除並回傳該 Value。

```python
score = student.pop("score")

print(score)
```

輸出

```text
100
```

---

# 查看所有 Key 與 Value

# get()

安全取得資料。

一般方式：

```python
print(student["phone"])
```

若不存在會發生：

```text
KeyError
```

使用 `get()`：

```python
print(student.get("phone"))
```

輸出

```text
None
```

設定預設值：

```python
print(student.get("phone", "沒有電話"))
```

輸出

```text
沒有電話
```

---

# in

檢查 Key 是否存在。

```python
if "age" in student:
    print("有 age")
```

---

# len()

取得 Dictionary 筆數。

```python
print(len(student))
```

---

# clear()

清空 Dictionary。

```python
student.clear()
```

---

# copy()

複製 Dictionary。

```python
student2 = student.copy()
```

---

# update()

一次更新多筆資料。

```python
student.update({
    "age": 20,
    "score": 100
})
```

---

# popitem()

刪除最後加入的一筆資料。

```python
student.popitem()
```

---

# setdefault()

若 Key 不存在才建立。

```python
student.setdefault("score", 60)
```

若已存在則不會覆蓋。

---

# 走訪 Dictionary

## 方法一：取得所有 Key

```python
for key in student:
    print(key)
```

---

## 方法二：Key + Value

```python
for key in student:
    print(key, student[key])
```

---

## 方法三（推薦）

```python
for key, value in student.items():
    print(key, value)
```

---

# Dictionary 常用方法整理

| 方法 | 功能 |
|------|------|
| `get()` | 安全取得 Value |
| `keys()` | 所有 Key |
| `values()` | 所有 Value |
| `items()` | 所有 Key 與 Value |
| `update()` | 更新多筆資料 |
| `clear()` | 清空 Dictionary |
| `copy()` | 複製 Dictionary |
| `pop()` | 刪除指定 Key |
| `popitem()` | 刪除最後一筆 |
| `setdefault()` | Key 不存在才新增 |
| `len()` | Dictionary 筆數 |

---

# List 與 Dictionary 比較

| List | Dictionary |
|------|------------|
| 使用 Index | 使用 Key |
| `student[0]` | `student["name"]` |
| 適合存放一系列資料 | 適合描述一個物件 |
| 查詢需知道位置 | 查詢只需知道 Key |

List：

```python
student = [
    "小明",
    18,
    95
]
```

Dictionary：

```python
student = {
    "name": "小明",
    "age": 18,
    "score": 95
}
```
