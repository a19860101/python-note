# Python tuple 基礎與方法

## 1. 什麼是 tuple？

`tuple` 中文常翻成 **元組**，是一種可以存放多個資料的容器。

基本寫法使用小括號 `()`：

```python
fruits = ("apple", "banana", "cherry")
print(fruits)
```

輸出：

```python
('apple', 'banana', 'cherry')
```

---

## 2. tuple 與 list 的差別

| 類型    | 符號   | 可不可以修改 | 適合用途            |
| ----- | ---- | ------ | --------------- |
| list  | `[]` | 可以修改   | 購物清單、成績表、會變動的資料 |
| tuple | `()` | 不可以修改  | 座標、固定設定、不想被改的資料 |

### list 可以修改

```python
fruits = ["apple", "banana"]
fruits[0] = "orange"
print(fruits)
```

輸出：

```python
['orange', 'banana']
```

### tuple 不可以修改

```python
fruits = ("apple", "banana")
fruits[0] = "orange"
```

錯誤訊息：

```python
TypeError: 'tuple' object does not support item assignment
```

---

## 3. 建立 tuple

### 一般 tuple

```python
colors = ("red", "green", "blue")
```

### 不加括號也可以建立 tuple

```python
colors = "red", "green", "blue"
print(colors)
```

輸出：

```python
('red', 'green', 'blue')
```

### 只有一個元素的 tuple

| 寫法           | 型別      | 說明               |
| ------------ | ------- | ---------------- |
| `("apple")`  | `str`   | 這是字串，不是 tuple    |
| `("apple",)` | `tuple` | 這才是只有一個元素的 tuple |

```python
data = ("apple")
print(type(data))
```

輸出：

```python
<class 'str'>
```

```python
data = ("apple",)
print(type(data))
```

輸出：

```python
<class 'tuple'>
```

---

## 4. 讀取 tuple 裡面的資料

tuple 可以用索引值讀取資料，索引值從 `0` 開始。

```python
fruits = ("apple", "banana", "cherry")

print(fruits[0])
print(fruits[1])
print(fruits[2])
```

輸出：

```python
apple
banana
cherry
```

---

## 5. 負索引

| 寫法   | 意思    |
| ---- | ----- |
| `-1` | 最後一個  |
| `-2` | 倒數第二個 |

```python
fruits = ("apple", "banana", "cherry")

print(fruits[-1])
print(fruits[-2])
```

輸出：

```python
cherry
banana
```

---

## 6. 切片 slicing

```python
fruits = ("apple", "banana", "cherry", "orange", "grape")

print(fruits[1:4])
```

輸出：

```python
('banana', 'cherry', 'orange')
```

| 寫法            | 說明                    |
| ------------- | --------------------- |
| `fruits[1:4]` | 從索引 `1` 開始，到索引 `4` 之前 |
| 包含            | 索引 `1`                |
| 不包含           | 索引 `4`                |

---

## 7. tuple 不能修改

tuple 是 **immutable**，也就是 **不可變的**。

以下操作都不能使用：

```python
fruits = ("apple", "banana", "cherry")

fruits[0] = "orange"
fruits.append("grape")
fruits.remove("banana")
```

| 操作         | tuple 是否支援 |
| ---------- | ---------- |
| 修改元素       | 不支援        |
| `append()` | 不支援        |
| `remove()` | 不支援        |
| `pop()`    | 不支援        |

---

## 8. tuple 常用方法

tuple 常用方法主要有兩個：

| 方法        | 說明            |
| --------- | ------------- |
| `count()` | 計算某個值出現幾次     |
| `index()` | 找出某個值第一次出現的位置 |

---

## 9. count()

`count()` 用來計算某個資料出現幾次。

```python
numbers = (1, 2, 3, 2, 4, 2)

print(numbers.count(2))
```

輸出：

```python
3
```

---

## 10. index()

`index()` 用來找出某個值第一次出現的位置。

```python
fruits = ("apple", "banana", "cherry")

print(fruits.index("banana"))
```

輸出：

```python
1
```

如果找不到資料，會出現錯誤。

```python
fruits = ("apple", "banana", "cherry")

print(fruits.index("orange"))
```

錯誤訊息：

```python
ValueError: tuple.index(x): x not in tuple
```

---

## 11. len()

`len()` 可以取得 tuple 的長度。

```python
fruits = ("apple", "banana", "cherry")

print(len(fruits))
```

輸出：

```python
3
```

---

## 12. 使用 in 判斷資料是否存在

```python
fruits = ("apple", "banana", "cherry")

print("apple" in fruits)
print("orange" in fruits)
```

輸出：

```python
True
False
```

也可以搭配 `if` 使用：

```python
fruits = ("apple", "banana", "cherry")

if "banana" in fruits:
    print("有香蕉")
else:
    print("沒有香蕉")
```

---

## 13. 使用 for 迴圈讀取 tuple

```python
fruits = ("apple", "banana", "cherry")

for fruit in fruits:
    print(fruit)
```

輸出：

```python
apple
banana
cherry
```

---

## 14. tuple unpacking (解包、解壓縮)

tuple 可以將資料拆開，分別放到不同變數中。

```python
person = ("John", 25, "Taiwan")

name, age, country = person

print(name)
print(age)
print(country)
```

輸出：

```python
John
25
Taiwan
```

### 解包數量要一致

```python
person = ("Zac", 25, "Taiwan")

name, age = person
```

這樣會出錯，因為左邊只有 2 個變數，但右邊有 3 個資料。

---

## 15. tuple 可以存不同型別

```python
student = ("Amy", 18, 85.5, True)

print(student)
```

tuple 裡面可以同時存放不同型別的資料。

| 資料      | 型別          |
| ------- | ----------- |
| `"Amy"` | 字串 `str`    |
| `18`    | 整數 `int`    |
| `85.5`  | 浮點數 `float` |
| `True`  | 布林值 `bool`  |

---

## 16. tuple 裡面可以放 list

tuple 本身不能修改，但如果 tuple 裡面放的是 list，list 裡面的內容仍然可以修改。

```python
data = ("Amy", [80, 90, 100])

data[1].append(95)

print(data)
```

輸出：

```python
('Amy', [80, 90, 100, 95])
```

---

## 17. tuple 轉 list

如果需要修改 tuple，可以先轉成 list，修改後再轉回 tuple。

```python
fruits = ("apple", "banana", "cherry")

fruits_list = list(fruits)
fruits_list.append("orange")

fruits = tuple(fruits_list)

print(fruits)
```

輸出：

```python
('apple', 'banana', 'cherry', 'orange')
```

流程：

| 步驟 | 說明            |
| -- | ------------- |
| 1  | tuple 轉成 list |
| 2  | 修改 list       |
| 3  | list 轉回 tuple |

## 18. 生活實務例子：座標資料

tuple 可以用來表示固定資料，例如地圖座標。

```python
location = (25.0330, 121.5654)

latitude, longitude = location

print(f"緯度：{latitude}")
print(f"經度：{longitude}")
```

輸出：

```python
緯度：25.033
經度：121.5654
```

---

## 19. 生活實務例子：學生資料

```python
student = ("Amy", 18, "Python班")

name, age, course = student

print(f"姓名：{name}")
print(f"年齡：{age}")
print(f"課程：{course}")
```

輸出：

```python
姓名：Amy
年齡：18
課程：Python班
```

## 20. 重點整理

| 重點            | 說明                               |
| ------------- | -------------------------------- |
| tuple 使用 `()` | 例如 `("apple", "banana")`         |
| tuple 不能修改    | 不能改值、不能 `append()`、不能 `remove()` |
| 可以用索引讀取       | 例如 `data[0]`                     |
| 可以用負索引        | 例如 `data[-1]`                    |
| 可以用切片         | 例如 `data[1:3]`                   |
| 常用方法          | `count()`、`index()`              |
| 適合固定資料        | 座標、設定、固定選項                       |
| 一個元素要加逗號      | 例如 `("apple",)`                  |
| 可以解包          | 例如 `x, y = point`                |
| 可以轉成 list 修改  | `list()`、`tuple()`               |

---

## 22. list 與 tuple 使用時機

| 情況            | 建議使用  |
| ------------- | ----- |
| 資料之後會增加、刪除、修改 | list  |
| 資料是固定的，不希望被改  | tuple |
| 購物清單          | list  |
| GPS 座標        | tuple |
| 成績紀錄會更新       | list  |
| 固定選項          | tuple |

```python
shopping_list = ["牛奶", "雞蛋", "麵包"]

gps_location = (25.0330, 121.5654)
```
