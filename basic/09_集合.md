# Python `set` 基礎與方法

## 1. `set` 是什麼？

| 特性     | 說明              |
| ------ | --------------- |
| 不重複    | 相同資料只會保留一份      |
| 沒有順序   | 輸出順序不一定和建立時相同   |
| 可修改    | 可以新增、刪除元素       |
| 適合資料比對 | 可做聯集、交集、差集、對稱差集 |

---

## 2. 建立 `set`

| 寫法                | 說明                     |
| ----------------- | ---------------------- |
| `{元素1, 元素2, 元素3}` | 建立有資料的集合               |
| `set()`           | 建立空集合                  |
| `set(可迭代資料)`      | 將 list、tuple、字串等轉成 set |

### 範例

```python
fruits = {"apple", "banana", "orange"}
print(fruits)
```

```python
fruits = set(["apple", "banana", "orange"])
print(fruits)
```

```python
empty_set = set()
print(empty_set)
```

---

## 3. 空集合注意事項

| 寫法      | 資料型態 |
| ------- | ---- |
| `set()` | set  |
| `{}`    | dict |

```python
empty_set = set()
empty_dict = {}

print(type(empty_set))
print(type(empty_dict))
```

---

## 4. `set` 不會有重複資料

```python
numbers = {1, 2, 3, 3, 4, 4, 5}
print(numbers)
```

輸出：

```python
{1, 2, 3, 4, 5}
```

---

## 5. `set` 沒有順序

```python
fruits = {"apple", "banana", "orange"}
print(fruits)
```

輸出順序不固定。

`set` 不能使用索引取值：

```python
fruits[0]  # 錯誤
```

# 6. 新增資料

## 6.1 `add()`：新增一個元素

| 方法            | 說明     |
| ------------- | ------ |
| `set.add(元素)` | 新增一個元素 |

```python
fruits = {"apple", "banana"}

fruits.add("orange")

print(fruits)
```

---

## 6.2 `update()`：一次新增多個元素

| 方法                  | 說明       |
| ------------------- | -------- |
| `set.update(可迭代資料)` | 一次新增多個元素 |

```python
fruits = {"apple", "banana"}

fruits.update(["orange", "grape"])

print(fruits)
```

也可以加入另一個 `set`：

```python
a = {1, 2}
b = {3, 4}

a.update(b)

print(a)
```

---

# 7. 刪除資料

## 7.1 `remove()`：刪除指定元素

| 方法               | 說明              |
| ---------------- | --------------- |
| `set.remove(元素)` | 刪除指定元素，元素不存在會報錯 |

```python
fruits = {"apple", "banana", "orange"}

fruits.remove("banana")

print(fruits)
```

元素不存在會報錯：

```python
fruits.remove("grape")  # 錯誤
```

---

## 7.2 `discard()`：刪除指定元素

| 方法                | 說明               |
| ----------------- | ---------------- |
| `set.discard(元素)` | 刪除指定元素，元素不存在不會報錯 |

```python
fruits = {"apple", "banana", "orange"}

fruits.discard("banana")
fruits.discard("grape")

print(fruits)
```

---

## 7.3 `pop()`：刪除一個元素

| 方法          | 說明        |
| ----------- | --------- |
| `set.pop()` | 刪除並回傳一個元素 |

```python
fruits = {"apple", "banana", "orange"}

item = fruits.pop()

print(item)
print(fruits)
```

---

## 7.4 `clear()`：清空集合

| 方法            | 說明   |
| ------------- | ---- |
| `set.clear()` | 清空集合 |

```python
fruits = {"apple", "banana", "orange"}

fruits.clear()

print(fruits)
```

輸出：

```python
set()
```

# 10. 集合運算

假設有兩組資料：

```python
a = {"apple", "banana", "orange"}
b = {"banana", "grape", "apple"}
```

---

## 10.1 聯集

聯集：取得兩個集合的所有元素，不重複。

| 寫法           | 說明 |
| ------------ | -- |
| `a.union(b)` | 聯集 |
| `a \| b`     | 聯集 |

```python
result = a.union(b)
print(result)
```

```python
result = a | b
print(result)
```

結果：

```python
{"apple", "banana", "orange", "grape"}
```

---

## 10.2 交集

交集：取得兩個集合都有的元素。

| 寫法                  | 說明 |
| ------------------- | -- |
| `a.intersection(b)` | 交集 |
| `a & b`             | 交集 |

```python
result = a.intersection(b)
print(result)
```

```python
result = a & b
print(result)
```

結果：

```python
{"apple", "banana"}
```

---

## 10.3 差集

差集：取得 `a` 有，但 `b` 沒有的元素。

| 寫法                | 說明 |
| ----------------- | -- |
| `a.difference(b)` | 差集 |
| `a - b`           | 差集 |

```python
result = a.difference(b)
print(result)
```

```python
result = a - b
print(result)
```

結果：

```python
{"orange"}
```

---

## 10.4 對稱差集

對稱差集：取得兩邊不重複的部分。

| 寫法                          | 說明   |
| --------------------------- | ---- |
| `a.symmetric_difference(b)` | 對稱差集 |
| `a ^ b`                     | 對稱差集 |

```python
result = a.symmetric_difference(b)
print(result)
```

```python
result = a ^ b
print(result)
```

結果：

```python
{"orange", "grape"}
```

---

# 11. 集合運算符號整理

| 符號   | 方法                       | 名稱   | 說明        |
| ---- | ------------------------ | ---- | --------- |
| `\|` | `union()`                | 聯集   | 全部合起來，不重複 |
| `&`  | `intersection()`         | 交集   | 兩邊都有      |
| `-`  | `difference()`           | 差集   | 左邊有，右邊沒有  |
| `^`  | `symmetric_difference()` | 對稱差集 | 兩邊不一樣的部分  |

---

# 12. 三組資料一起運算

假設有三組資料：

```python
a = {"小明", "小美", "阿強"}
b = {"小美", "阿強", "小華"}
c = {"阿強", "小安", "小美"}
```

---

## 12.1 三組資料做聯集

聯集：三組資料全部合起來，不重複。

| 寫法              | 說明     |
| --------------- | ------ |
| `a \| b \| c`   | 三組資料聯集 |
| `a.union(b, c)` | 三組資料聯集 |

```python
result = a | b | c
print(result)
```

```python
result = a.union(b, c)
print(result)
```

結果：

```python
{'小明', '小美', '阿強', '小華', '小安'}
```

---

## 12.2 三組資料做交集

交集：三組資料都有的元素。

| 寫法                     | 說明     |
| ---------------------- | ------ |
| `a & b & c`            | 三組資料交集 |
| `a.intersection(b, c)` | 三組資料交集 |

```python
result = a & b & c
print(result)
```

```python
result = a.intersection(b, c)
print(result)
```

結果：

```python
{'小美', '阿強'}
```

---

## 12.3 三組資料做差集

差集：`a` 有，但是 `b` 和 `c` 都沒有的元素。

| 寫法                   | 說明                 |
| -------------------- | ------------------ |
| `a - b - c`          | `a` 有，但 `b`、`c` 沒有 |
| `a.difference(b, c)` | `a` 有，但 `b`、`c` 沒有 |

```python
result = a - b - c
print(result)
```

```python
result = a.difference(b, c)
print(result)
```

結果：

```python
{'小明'}
```

---

## 12.4 三組資料做對稱差集

| 寫法                                                  | 說明         |
| --------------------------------------------------- | ---------- |
| `a ^ b ^ c`                                         | 一組一組進行對稱差集 |
| `a.symmetric_difference(b).symmetric_difference(c)` | 一組一組進行對稱差集 |

```python
result = a ^ b ^ c
print(result)
```

# 13. 用括號控制運算順序

集合運算可以使用括號控制順序。

假設：

```python
a = {1, 2, 3, 4}
b = {3, 4, 5}
c = {2, 4, 6}
```

---

## 13.1 `(a - b) & c`

| 步驟 | 運算           | 結果       |
| -- | ------------ | -------- |
| 1  | `a - b`      | `{1, 2}` |
| 2  | `{1, 2} & c` | `{2}`    |

```python
result = (a - b) & c
print(result)
```

結果：

```python
{2}
```

---

## 13.2 `a - (b & c)`

| 步驟 | 運算        | 結果          |
| -- | --------- | ----------- |
| 1  | `b & c`   | `{4}`       |
| 2  | `a - {4}` | `{1, 2, 3}` |

```python
result = a - (b & c)
print(result)
```

結果：

```python
{1, 2, 3}
```

---

## 13.3 括號運算比較

| 寫法            | 說明                    | 結果          |
| ------------- | --------------------- | ----------- |
| `(a - b) & c` | 先算 `a - b`，再與 `c` 取交集 | `{2}`       |
| `a - (b & c)` | 先算 `b & c`，再從 `a` 扣掉  | `{1, 2, 3}` |

# 15. 生活化範例：三天上課名單

```python
day1 = {"小明", "小美", "阿強"}
day2 = {"小美", "阿強", "小華"}
day3 = {"阿強", "小安", "小美"}

print("三天有來過的人：", day1 | day2 | day3)
print("三天都有來的人：", day1 & day2 & day3)
print("第一天有來，但第二、三天沒來的人：", day1 - day2 - day3)
```

輸出：

```python
三天有來過的人： {'小明', '小美', '阿強', '小華', '小安'}
三天都有來的人： {'小美', '阿強'}
第一天有來，但第二、三天沒來的人： {'小明'}
```

# 16. 生活化範例：用括號控制條件

```python
class_a = {"小明", "小美", "阿強", "小華"}
club_b = {"阿強", "小華", "小安"}
contest_c = {"小美", "小華", "小傑"}

result = (class_a - club_b) & contest_c

print(result)
```

結果：

```python
{'小美'}
```

意思：

| 條件                               | 說明                       |
| -------------------------------- | ------------------------ |
| `class_a - club_b`               | 在 A 班，但不在 B 社團           |
| `(class_a - club_b) & contest_c` | 在 A 班、不在 B 社團，而且有參加 C 比賽 |

# 17. 去除重複資料

```python
names = ["小明", "小美", "小明", "阿強", "小美", "小安"]

unique_names = set(names)

print(unique_names)
```

結果：

```python
{'小明', '小美', '阿強', '小安'}
```

轉回 list：

```python
unique_names = list(set(names))

```

# 18. 簡易記法

| 寫法            | 意思                    |
| ------------- | --------------------- |
| `a \| b`      | 全部合起來                 |
| `a & b`       | 兩邊都有                  |
| `a - b`       | `a` 有，`b` 沒有          |
| `a ^ b`       | 兩邊不一樣的部分              |
| `a \| b \| c` | 三組全部合起來               |
| `a & b & c`   | 三組都有                  |
| `a - b - c`   | `a` 有，但 `b`、`c` 沒有    |
| `(a - b) & c` | 先算 `a - b`，再與 `c` 取交集 |
