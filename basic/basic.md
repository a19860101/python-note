## 變數

### 變數命名規則

- 變數命名規則
- 變數名必須以字母或底線開頭。
- 變數名不能以數字開頭。
- 變數名稱只能包含字母數字字元和底線（Az、0-9 和 _）。
- 變數名稱區分大小寫（age、Age 和 AGE 是三個不同的變數）。
- 變數名不能是任何Python 關鍵字。

```python
name = 'Zac'
print(name)
```

| 型別      | 說明      | 範例              |
| ------- | ------- | --------------- |
| `str`   | 字串| `"小明"`          |
| `int`   | 整數      | `18`            |
| `float` | 浮點數      | `3.14`          |

### 檢查型別

可以使用`type()`檢查型別

```python
name = "小明"
age = 18
height = 170.5
is_student = True

print(type(name))
print(type(age))
print(type(height))
print(type(is_student))

```

輸出：

```bash
<class 'str'>
<class 'int'>
<class 'float'>
<class 'bool'>
```

## 註解

可分為單行註解與多行註解，不會被程式執行，是用來解釋程式碼，提高可讀性的方式。

```python
# 我是單行註解

"""
我是多行註解
"""
'''
我也是多行註解
'''
```

嚴格說，他其實不是註解，只是多行字串。

| 寫法        | 實際意義  | 常見用途                |
| --------- | ----- | ------------------- |
| `#`       | 真正的註解 | 單行註解                |
| `''' '''` | 多行字串  | 暫時註解一大段、寫多行文字       |
| `""" """` | 多行字串  | docstring、函式說明、類別說明 |

## 運算子

用來「計算」或「比較」資料的符號。

### 算術運算子

常用的數學運算

| 運算子  | 說明  | 範例        | 結果          |
| ---- | --- | --------- | ----------- |
| `+`  | 加法  | `10 + 3`  | `13`        |
| `-`  | 減法  | `10 - 3`  | `7`         |
| `*`  | 乘法  | `10 * 3`  | `30`        |
| `/`  | 除法  | `10 / 3`  | `3.3333...` |
| `//` | 整除  | `10 // 3` | `3`         |
| `%`  | 取餘數 | `10 % 3`  | `1`         |
| `**` | 次方  | `10 ** 3` | `1000`      |

### 比較運算子

| 運算子  | 說明    | 範例         | 結果      |
| ---- | ----- | ---------- | ------- |
| `==` | 是否相等  | `10 == 10` | `True`  |
| `!=` | 是否不相等 | `10 != 3`  | `True`  |
| `>`  | 大於    | `10 > 3`   | `True`  |
| `<`  | 小於    | `10 < 3`   | `False` |
| `>=` | 大於等於  | `10 >= 10` | `True`  |
| `<=` | 小於等於  | `3 <= 10`  | `True`  |

### 指定運算子

| 運算子  | 說明    | 範例       | 等同於         |
| ---- | ----- | -------- | ----------- |
| `=`  | 指定    | `x = 10` | 把 10 放進 x   |
| `+=` | 加完再指定 | `x += 1` | `x = x + 1` |
| `-=` | 減完再指定 | `x -= 1` | `x = x - 1` |
| `*=` | 乘完再指定 | `x *= 2` | `x = x * 2` |
| `/=` | 除完再指定 | `x /= 2` | `x = x / 2` |

### 邏輯運算子

| 運算子   | 說明 |
| ----- | -- |
| `and` | 而且 |
| `or`  | 或者 |
| `not` | 相反 |

### 成員運算子

text = "I love Python"
